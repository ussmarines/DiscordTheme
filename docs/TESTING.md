# Testing

## Local file test

After pushing the repo, install:

```text
themes/sibylla.theme.css
```

The installable theme imports:

```text
https://raw.githubusercontent.com/ussmarines/DiscordTheme/main/build/sibylla.css
```

So the repo must be pushed before the raw URL can load the latest build.

## Online Vencord test

Use this URL:

```text
https://raw.githubusercontent.com/ussmarines/DiscordTheme/main/themes/sibylla.theme.css
```

npm run verify
