## install

### vencord / betterdiscord

1. download [`sibnight.theme.css`](https://github.com/ussmarines/DiscordTheme/blob/main/themes/sibnight.theme.css)
2. drag the file into your theme folder
3. customize the variables directly in `sibnight.theme.css`

### install through link

add `https://raw.githubusercontent.com/ussmarines/DiscordTheme/main/build/sibnight.css` to your theme import links.

## contributing

1. clone the repository
2. run `npm i`
3. create a `.env` file in the project root

```env
DEV_OUTPUT_PATH=C:\Users\USERNAME\AppData\Roaming\Vencord\themes\sibnight-dev.theme.css
```

4. run `npm run dev`
5. edit files in `/src` or `themes/sibnight.theme.css`
6. commit both `/src` and `/build` changes together
